Hijacking Search Engines

Hijacking a search engines is simple enough and only requires basic knowledge of HTML and how a web URL works.

A web URL is a file path that is constructed in a way so that each link references a file within a folder. So when you click a link and a different webpage opens up that is because the link is referencing that file on the server.

The search function can be a file or page to itself but usually it's just an element within a file and can be written in the URL.

That is how these search engines will be Hijacked. The syntax primarily is "Google.com/search" or "Reddit.com/search"

By placing this web address in the form action element of my HTML document will start the web address. Whatever text that is typed into the search bar will be amended to the form action along with the "input" elements name resulting in a full search URL address for the website to be able to use.
